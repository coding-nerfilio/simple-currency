# simple-currency

Currency exchanger PWA powered by [CurrencyConverterApi.com](https://CurrencyConverterApi.com).
Visit it on [https://simple-currency-pwa.web.app/](https://simple-currency-pwa.web.app/).

## Motivations

Wanted to have a small and handy PWA for fast and reliable currency conversions from any device, it also works as a brief
showcase of some skills like API fetching, localStorage management, Redux and type safety.

## Playing around

Just git clone, and install dependencies:

```
git clone https://github.com/coding-nerfilio/simple-currency
cd simple-currency
yarn install
```

Don't forget to add your own CurrencyConverter API key to .env file (root folder):

```
REACT_APP_CURRENCYCONVERTER_API_KEY = YOUR_API_KEY_HERE
```

You are ready to go!

```
yarn start
```
